# StyleSense AI 👗✨

StyleSense AI is an AI-powered fashion advisor that helps users discover outfits, match colors, receive styling recommendations, and build confidence through personalized fashion guidance.

## 🚀 Features

- 👕 Personalized outfit recommendations
- 🎨 Color palette analysis
- 👗 Occasion-based styling suggestions
- 👜 Clothing and accessory matching
- 💄 Makeup and hairstyle recommendations
- 📱 Responsive and modern UI
- 🤖 AI-powered fashion assistance

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- Gemini API
- Node.js
- Vite

## 📂 Project Structure

```
src/
components/
pages/
assets/
public/
```

## ⚙️ Installation

```bash
git clone <repository-url>
cd stylesense-ai
npm install
npm run dev
```
